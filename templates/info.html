Backend Ishlar Ro'yxati

Ma'lumotlar Bazasi Modelli va Arxitekturasi

  • Foydalanuvchilar: Profil, shaxsiy ma'lumotlar, parolni tiklash, do'stlar ro'yxati.
  • Sayohatlar: Sayohat nomi, tavsifi, shaharlar, sanalar, qo'shilgan do'stlar.
  • Obunalar: Obuna turlari va foydalanuvchilarning obuna holati.
  • Xaritalar: Geolokatsiya orqali joylashuvlar va tashrif belgilari.
  • Valyuta ma'lumotlari: API orqali yangilanadigan valyuta kurslari.
  • Kalkulyatorlar: Sayohat va yashash xarajatlari uchun hisoblash parametrlarini saqlash.
  • Statik ma'lumotlar: Mamlakatlar, shaharlar va yashash darajasi haqidagi statik ma'lumotlar.

API Interfeysi

  • Ro'yxatdan o'tish va avtorizatsiya: JWT yordamida autentifikatsiya.
  • Foydalanuvchi profili: Ma'lumotlarni ko'rish va tahrirlash, do'stlarni boshqarish.
  • Sayohatlar: Yaratish, o'zgartirish, o'chirish, kelajak va o'tgan sayohatlarni ajratish.
  • Xaritalar va geolokatsiya: Foydalanuvchining joylashuvini aniqlash, tashrif joylarini saqlash.
  • Valyuta konvertatsiyasi: API orqali valyuta kurslarini olish va hisoblash.
  • Kalkulyatorlar: Sayohat va yashash xarajatlarini hisoblash API'lari.
  • Bildirishnomalar: Yangi sayohat, obuna yoki do'st qo'shilganida bildirishnomalar yuborish.

Integratsiyalar

  • Valyuta API: Exchange Rates API orqali valyuta kurslarini olish.
  • Geolokatsiya API: Google Maps yoki OpenStreetMap API integratsiyasi.
  • To'lov tizimi: Obunalar uchun Yukassa kabi to'lov tizimi integratsiyasi.
  • Bildirishnoma tizimi: FCM yoki APNs orqali mobil bildirishnomalar.

Xavfsizlik

  • Token boshqaruvi: JWT tokenlarni yangilash va qora ro'yxatga olish.
  • Ma'lumotlarning shifrlanishi: Parollarni shifrlash (bcrypt yoki argon2).
  • API xavfsizligi: Rate limiting, API kalitlari.
  • HTTPS ulanish: Ma'lumotlarni xavfsiz uzatish.

Admin Paneli

  • Foydalanuvchilarni boshqarish.
  • Sayohatlar, obunalar va boshqa ma'lumotlarni kuzatish.
  • Statik ma'lumotlarni boshqarish.

Texnologiyalar va Ramkalar

  • Til va platforma: Node.js, Express.js yoki Nest.js ramkasi.
  • Ma'lumotlar ombori: MySQL.
  • Ma'lumotlar keshlash: Redis.
  • Testlash: Postman yoki Swagger yordamida API testlash.

Sinov va Disk Raskadrovka

  • Backend funksionalligini to'liq testlash.
  • Har bir API uchun unit test va integratsion testlar.
  • Foydalanuvchi profillari, sayohatlar, kalkulyatorlar va obunalar uchun stress testlar.
